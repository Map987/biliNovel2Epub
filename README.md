# 使用方法1 
1. 安装Pyhon3 
2. 执行`pip3 install -r requirement.txt`安装依赖 
3. 执行`python3 bilinovel2epub.py <ID>`, `<ID>`替换为书籍ID或书籍首页链接 

例子：`python3 bilinovel2epub.py https://w.linovelib.com/novel/2611.html`  
运行成功后会在程序运行目录下找到对应的Epub文件
